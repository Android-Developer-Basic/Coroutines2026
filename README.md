# Демо-приложение для лекции Coroutines (Basic)

Репозиторий содержит [Android-приложение](coroutines) c демонстрацией:

- эмуляции прямого обращения к сети из основного потока
- отправки запроса из фонового потока
- попытки централизованной обработки ошибки
- переделанного приложения, с использованием Coroutines

Примеры работы с библиотекой разобраны в [отдельном модуле](coroutines/experiments).
Все примеры разложены по шагам презентации.
